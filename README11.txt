Program Purpose:
My project utilizes a photoreceptor, white LED, and piezo speaker.
The photoreceptor finds the value of the light level, which the analog
signal of that is used for the piezo speaker frequency.
The white LED provides light in dark places.

How to Use:
1. Unzip the project file
2. Double-click lightheremin.ino to start the Arduino
3. Set up circuit
3. Press the Upload button
